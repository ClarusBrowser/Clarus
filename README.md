# Bienvenue sur Clarus !
-----
## C'est quoi Clarus ?
Clarus est un navigateur web français qui est open-source et permet de le personnaliser comme souhaité.

## Et, les fonctions ?
* Clarus ne contient aucune fonction IA
* Clarus contient beaucoup de fonction de personnalisation
* Affiche pratiquement tout les sites !

# Installation
Pour installer, il faut :
1. Accéder à la page de "Releases"
2. Sélectionner la version souhaitée
3. Appuyer sur le fichier EXE
4. Lancer l'installation
5. Utiliser l'application !

>[!CAUTION]
>Il est possible que Windows dit que Clarus est un virus, un fichier malveillant ou qu'il n'y ai pas d'éditeur certifié.
>
>C'est normal, étant donné que le projet est récent et qu'il faut payer énormément pour ajouter un éditeur, il est possible que ça arrive.
